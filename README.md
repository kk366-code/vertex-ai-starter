`uv` を使用して、Google Cloud Vertex AI (Gemini) を利用するためのスタータープロジェクトです。

## 🚀 特徴

- **最新のSDK**: `google-genai` を使用
- **モダンなパッケージ管理**: `uv` による高速で再現可能な環境構築
- **プロキシ/制限環境対応**: gRPC制限を回避するHTTP/1.1接続設定済み
- **環境変数の分離**: `python-dotenv` による安全な設定管理
